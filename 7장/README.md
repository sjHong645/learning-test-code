파이썬 코드의 구조를 일부 개선한다. 
- 모듈로 `프로덕션 코드`에서 `테스트 코드를 분리`한다.
- 파이썬의 범위(scope)와 import 규칙이 어떻게 코드 내 의존성을 올바르게 보장하는지 살펴본다.
- 코드에서 중복 테스트를 제거해서 테스트를 보다 조밀하고 의미 있게 만들어본다
